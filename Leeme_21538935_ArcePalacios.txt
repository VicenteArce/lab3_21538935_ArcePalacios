Si falla su compilacion o ejecuación, este proyecto fue relizado en JDK 11 usando gradle en la version 8.2.

Primeramente asegurese de contar con todos los archivos del proyecto en una misma carpeta.
Luego abra una terminal/consola en el directorio principal del proyecto y ejecute uno de los siguientes comandos dependiendo de su sistema operativo:
* para compilar: ./gradlew build | para ejecutar: ./gradlew run (en GNU/Linux)
* para compilar: gradlew.bat build | para ejecutar: gradlew.bat run (en Windows)

Se sugiere que primero se prueben los requisitos del usuario comun con el sistema precargado, ya que si usted modifica el sistema, puede que deje de funcionar correctamente, ya que se podria referenciar mal algunos códigos al momento de crear una opcion, un flujo, un chatbot o incluso un sistema.
